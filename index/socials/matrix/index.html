What is Matrix?

Matrix is a decentralized communication method. Due to its decentralized nature its best to create accounts on homeserver that aren't the official one. I created my own instance of the Matrix Synapse server software using docker and ansible. If you would like to create an account on a different server, you can check out some here. I recommend using the Element client, although others can still be used too.

You can create an account on my homeserver using this link: https://matrix.arrayinamatrix.xyz
