Privacy Policy: Matrix

last updated: 15/01/2022
We record server analytics and statistics i.e. information specific to the matrix synapse server (how long it takes to send a message, how many times Synapse attempts to retrieve data from a cache, etc.). Server analytics are only used to improve and monitor the health of the Synapse server. We only require a password and username to use our matrix chat server, all other data (email address, phone number, etc.) is optional. Server admin(s) have access all unencrypted data (Please use end to end encryption.)! We report anonymous telemetry to matrix.org . The information reported can be found here.

Rules for our matrix homeserver can be found here.
